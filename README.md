## 📋 What is Smart-Energy-Meter-Management?

Smart-Energy-Meter-Management is a system that uses an ESP32 device to track real-time voltage, current, power, and motion data. It shows this information on an LCD screen and lets you see it remotely through a web dashboard and Blynk app. The system helps you use energy more efficiently and detect any unusual activity. It works well in places like labs, hostels, classrooms, and canteens.

This application lets you keep track of energy use easily, even if you are not a technical person. You can view live readings and history to spot areas where you can save power and avoid problems early.

## 🌐 How it Works

The system connects to an ESP32 microcontroller. This device measures electricity usage and motion in real time. The device sends data to your Windows PC, which then displays it locally on an LCD and remotely through a web dashboard or the Blynk app.

You do not need to know how to program. The software will guide you through connecting the ESP32 and opening the dashboard on your browser.

## 🔧 Troubleshooting

### ESP32 Not Detected

- Check if the USB cable is firmly connected.

- Try a different USB port on your PC.

- Restart the software and reconnect the device.

- Make sure drivers for ESP32 are installed (the software may prompt you).

### Software Won’t Open

- Confirm your Windows version meets requirements.

- Restart your computer and try again.

- Reinstall the software if the problem continues.

### Web Dashboard Not Loading

- Verify your PC and ESP32 are on the same network.

- Check the address shown in the program for accuracy.

- Refresh your browser or try a different one.
